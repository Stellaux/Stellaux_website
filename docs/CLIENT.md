# Stellaux — Client (Storefront) Specifications

> Page list, actors, and overall scope live in [REQUIREMENTS.MD](REQUIREMENTS.MD).
> This document covers storefront-specific design notes per page.

The main website has the following required pages:
home page, store page, story page, craft page, and contact page.

## Home page

The home page features a hero-style carousel displaying the hottest trending items, with
slow-scroll transition animations between items. Beneath it is a smaller carousel
displaying the hot items.

## Current state

- The e-commerce (store) section is mostly set, with explicit modifications pending on the input fields.
- The craft page requires heavy restructuring and modification.
